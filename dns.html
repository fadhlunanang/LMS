DNS SERVER

Bagian ini bertujuan sebagai penguatan dalam memahami konsep DNS Server. DNS Server merupakan salah satu layanan penting yang berfungsi menerjemahkan domain ke IP maupun sebaliknya. Yuk simak materi berikut!

Silahkan mengerjakan latihan berikut dengan panduan yang ada!

Pada latihan ini anda akan mengonfigurasi DNS Server!

OUTPUT

Membuat DNS Server sukses mentranslasikan IP ke domain sesuai dengan ketentuan yang ada

Prepare the Lab

Sebelum anda memulai lab, anda dapat meminta akses ke guru anda. Akses akan berupa ssh -p [port] user@ipaddress


Jika sudah, anda bisa memulai lab dengan mengetikan perintah

lab start dns

Instructions

1. Masuk ke dalam mode root

user@server:~$ sudo su
[sudo] password for user:

2. Install paket DNS

root@server:~# apt-get install bind9 dns-utils -y

3. Hidupkan paket DNS yang sudah terinstall. (Pada dasarnya DNS langsung aktif ketika diinstall)

root@server:~# systemctl enable bind9

4. Masuk ke dalam direktori bind.

root@server:~# cd /etc/bind

5. Copy file forward dan reverse zone untuk membuat file baru.

root@server:~# cp db.local db.exercise | cp db.127 db.reverse.exercise

6. Isi file zone utama dengan perintah nano named.conf.local

zone "webme.com" {
type master;
file "/etc/bind/db.exercise";
};
zone ".....in-addr.arpa" {
type master;
file "/etc/bind/db.reverse.exercise";
};

7. Tambahkan baris untuk memasukkan IP agar dapat mentranslasikan subdomain www.webme.com pada file forward zone yang baru. Ketik dengan perintah nano db.exercise

.....
@ IN A your_ip
www IN A your_ip
.....

8. Tambahkan baris untuk memasukkan subdomain www.webme.com agar dapat mentranslasikan IP pada file reverse zone yang baru. Ketik dengan perintah nano db.reverse.exercise

.....
15 IN PTR webme.com.
15 IN PTR www.webme.com.
.....

9. Tambahkan nameserver dengan IP yang ada di adapter kedua dan domain yang sudah ada pada file resolv.conf agar DNS Client mengetahui DNS Server. Ketik dengan perintah nano /etc/resolv.conf

.....
nameserver your_ip
search webme.com
.....

10. Restart paket bind9

root@server:~# systemctl restart bind9

11. Lakukan pengecekkan dengan perintah berikut

root@server:~# nslookup webme.com
root@server:~# nslookup your_ip

Evaluation

Jika konfigurasi sudah selesai dilakukan, silahkan cek hasil pengerjaan dengan perintah di bawah ini. Jika dirasa belum cukup, anda dapat melanjutkan pengerjaan sampai hasilnya sudah benar

root@server:~# lab grade dns

Finish

Lakukan perintah berikut untuk menyelesaikan proses praktikum. Gunakan user root untuk melakukan perintah.

root@server:~# lab finish dns

Kerjakan latihan berikut untuk menyelesaikan permasalahan DNS Server.
