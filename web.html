WEB SERVER MENGGUNAKAN APACHE

Bagian ini bertujuan sebagai penguatan dalam memahami konsep Web Server khususnya Apache. Seperti yang kita tahu, Apache merupakan salah satu jenis Web Server yang mudah dikonfigurasi dan memiliki banyak fitur yang bermanfaat. Yuk simak materi di bawah ini!

Silahkan mengerjakan latihan berikut dengan panduan yang ada!

Pada latihan ini anda akan mengonfigurasi Web Server Apache!

OUTPUT

Membuat Apache sukses menampilkan website secara lokal

Prepare the Lab

Sebelum anda memulai lab, anda dapat meminta akses ke guru anda. Akses akan berupa ssh -p [port] user@ipaddress


Jika sudah, anda bisa memulai lab dengan mengetikan perintah

lab start apache

Instructions

1. Masuk ke dalam mode root

user@server:~$ sudo su
[sudo] password for user:

2. Install paket Apache

root@server:~# apt-get install apache2 -y

3. Hidupkan paket apache yang sudah terinstall. (Pada dasarnya apache langsung aktif ketika diinstall)

root@server:~# systemctl enable apache2

4. Buat direktori baru dalam /var/www/html untuk menyimpan file html custom.

root@server:~# mkdir /var/www/html/web-me

5. Buat file html baru di dalam direktori yang sudah dibuat yakni /var/www/html/web-me/.

root@server:~# touch /var/www/html/web-me/index.html

6. Isi file tersebut dengan Selamat datang di website

root@server:~# echo "<h1>Selamat datang di website</h1>" > /var/www/html/web-me

7. Pindah ke dalam direktori konfigurasi apache yang didalamnya terdapat file virtualhost

root@server:~# cd /etc/apache2/sites-available

8. Buat file virtualhost baru pada direktori tersebut yang diberi nama web-me.conf. Masuk ke dalam file tersebut dengan perintah nano web-me.conf

<VirtualHost *:80>
ServerName webme.com
ServerAdmin sija@localhost
DocumentRoot /var/www/html/web-me

<Directory /var/www/html/web-me>
Options Indexes FollowSymLinks
Require all granted
AllowOverride None
</Directory>
</VirtualHost>

9. Kita bisa menghidupkan hasil dari file virtualhost yang sudah kita buat dengan perintah

root@server:~# a2ensite web-me.conf

10. Matikan file virtualhost default yang ada pada direktori sites-enabled

root@server:~# a2dissite 000-default.conf

11. Restart paket apache

root@server:~# systemctl restart apache2

Evaluation

Jika konfigurasi sudah selesai dilakukan, silahkan cek hasil pengerjaan dengan perintah di bawah ini. Jika dirasa belum cukup, anda dapat melanjutkan pengerjaan sampai hasilnya sudah benar

root@server:~# lab grade apache

Finish

Lakukan perintah berikut untuk menyelesaikan proses praktikum. Gunakan user root untuk melakukan perintah.

root@server:~# lab finish apache

Kerjakan latihan berikut untuk menyelesaikan permasalahan Apache.

Prepare the Lab

Jika sudah, anda bisa memulai lab dengan mengetikan perintah

lab start solveApache

Clue

1. Masuk ke dalam mode root

2. Terdapat file index.html di dalam direktori /var/www/html/lab

3. Terdapat file VirtualHost dengan nama lab.conf

4. Terdapat domain dengan nama lab.com

5. File VirtualHost tersebut harus aktif dan sudah sesuai isinya seperti, server name, document root, hak akses, dan lain-lain.

6. Pastikan port listen sudah sesuai dengan yang ada pada file VirtualHost

Evaluation
